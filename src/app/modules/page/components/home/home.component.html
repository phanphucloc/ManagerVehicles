Manager vehicles


Description
  • This is a small web signal page built by framework angular version 9.1.9.
  • Provides basic features such as for users to view data lists, search, view details.

Feature
  • View shows vehicle list data from the StarWars-Api.
  • The modal displays vehicles' detailed data from the StarWars-Api.
  • Search for vehicles by name from the StarWars-Api.
